## 如何测试播放功能

### 方法1：使用API上传音频文件

1. 准备一些MP3音频文件
2. 使用Postman或类似工具发送POST请求：
   ```
   POST http://localhost:3001/api/songs
   Content-Type: multipart/form-data
   
   Fields:
   - audio: [你的音频文件]
   - title: "歌曲名称"
   - artist: "艺术家"
   ```

### 方法2：手动添加音频文件

1. 在 `backend/uploads/` 目录放入音频文件
2. 使用Prisma Studio手动添加数据库记录：
   ```bash
   cd backend
   npx prisma studio
   ```
3. 在Song表中添加记录，`filePath`指向你的文件

### 方法3：创建文件上传页面（推荐）

我可以帮你创建一个简单的上传页面，让你可以通过浏览器上传音频文件。
